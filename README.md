## Examples

Here are a few examples of shapes you can create using this tutorial:

### Circle

```python
import turtle

def draw_circle(radius):
    turtle.circle(radius)

draw_circle(50)
turtle.done()
```

### Square

```python
import turtle

def draw_square(side_length):
    for _ in range(4):
        turtle.forward(side_length)
        turtle.right(90)

draw_square(100)
turtle.done()
```

### Triangle

```python
import turtle

def draw_triangle(side_length):
    for _ in range(3):
        turtle.forward(side_length)
        turtle.left(120)

draw_triangle(100)
turtle.done()
```
